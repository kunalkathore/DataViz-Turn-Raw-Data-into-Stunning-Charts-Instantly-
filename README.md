# 📊 DataViz — Turn Your Data Into Beautiful Charts

DataViz is a modern web application that helps you convert raw data into stunning, interactive charts in seconds. Upload your dataset, choose a chart type, and export high-quality visuals effortlessly.

---

## 🚀 Features

✨ Upload CSV / dataset files  
📈 Generate multiple chart types (Bar, Line, Pie, etc.)  
⚡ Fast and interactive data visualization  
🎨 Clean and user-friendly UI  
📤 Export charts as images  
📱 Responsive design (works on desktop & mobile)
